# Copilot Dashboard

Copilot Dashboard — 18 LLM Provider Integration.

## Features

- 18 LLM provider integrations (OpenAI, Anthropic, Google, OpenRouter, etc.)
- Iceberg prompt engine for context management
- Provider switching and fallback configuration
- Token usage tracking and rate limiting
- Multi-model chat interface

## Tech Stack

Node.js 18+ · JavaScript ES6+ · WebSocket · MCP Server

## Quick Start

```bash
git clone https://github.com/D-VKITZ/copilot-dashboard.git
cd copilot-dashboard
npm install
npm run dev
```
