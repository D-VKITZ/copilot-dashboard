## Features

| Feature | Description |
|:--------|:------------|
| AI Copilot | Context-aware assistant for the DEVKiTZ ecosystem |
| 18 LLM Providers | Gemini, Claude, GPT, DeepSeek, Qwen, Mistral and more |
| Iceberg Prompt Engine | Structured prompt system with context injection |
| Chat Tab | Conversational interface with markdown rendering |
| Builder Tab | Visual prompt builder with template support |
| Settings Tab | Provider configuration, API keys and preferences |

---

## Tech Stack

| Technology | Purpose |
|:-----------|:--------|
| HTML5 | Tab-based layout with semantic structure |
| CSS3 | DkZ Design System with red-purple theme |
| JavaScript ES6+ | LLM provider abstraction and chat engine |
| Marked.js | Markdown rendering in chat responses |
| LocalStorage | Conversation history and settings persistence |
